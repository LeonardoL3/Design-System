# Vortex UI

Personal design system built with Storybook and React. Used as a shared component library across personal projects.

**Live:** https://leonardol3.github.io/Design-System

## Stack

- **Components:** React + TypeScript
- **Docs:** Storybook
- **Bundler:** Turbopack
- **CI/CD:** GitHub Actions (auto-deploys to GitHub Pages)

## Getting Started

```bash
npm install
npm run dev
npm run storybook
```

## Deployment

Merging to `main` triggers a GitHub Actions workflow that builds Storybook and deploys to GitHub Pages automatically.
